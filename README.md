# Yaswanth Reddy - Portfolio

Personal portfolio site showcasing my work as an AI/ML Engineer, built with vanilla HTML, CSS, and JavaScript.

## About

I'm an AI/ML Engineer at Spurring Ventures, Bengaluru, working on production computer vision and GenAI systems from YOLO11 detection pipelines to LangGraph-orchestrated agentic workflows.

## Features

- Responsive, accessible single-page layout with light/dark theme toggle
- Sections: About, Experience, Projects, Skills, Education, Contact
- Live blog feed pulled from Medium via RSS
- Built with no frameworks plain HTML/CSS/JS

## Tech Stack

- HTML5, CSS3, vanilla JavaScript
- Deployed via GitHub Pages

## Local Setup

Clone the repo and open `index.html` in a browser — no build step required.

\`\`\`bash
git clone https://github.com/yaswanthreddy3/portfolio.git
cd portfolio
open index.html
\`\`\`
